CentOS7安装和配置jre1.8

发表于2021-06-17|更新于2021-08-04|CentOS7
|字数总计:324|阅读时长:1分钟|阅读量:

下载jre

①官网下载地址：Server JRE (Java SE Runtime Environment) 8 Downloads

②百度网盘链接

链接：https://pan.baidu.com/s/1aRzE9hdak3N_KtkYMYyRKQ

提取码：mbv3

使用FTPRush上传jre到/home/data目录

详细步骤查看博客：FTPRush向CentOS7上传文件和从CentOS7下载文件

解压以及剪切到/home/java/目录

①创建/home/java目录

[root@localhost ~]# cd /
[root@localhost /]# cd home
[root@localhost home]# mkdir java

②解压tar.gz压缩包

[root@localhost data]# tar -zxvf server-jre-8u131-linux-x64.tar.gz

③把/home/data/下的jre剪切到/home/java/下

[root@localhost data]# mv /home/data/jdk1.8.0_131 /home/java

④给jre目录以及子目录授予root权限

[root@localhost data]# chown root:root -R /home/java/jdk1.8.0_131/

配置环境变量

①编辑环境变量配置文件

[root@localhost data]# vi /etc/profile

在文件末尾加入下面代码（jdk名称根据实际情况填写），强制保存（wq!）退出

export JAVA_HOME=/home/java

export JRE_HOME=/home/java/jdk1.8.0_131

export CLASSPATH=$JRE_HOME/lib/rt.jar:$JRE_HOME/lib/ext

export PATH=$PATH:$JRE_HOME/bin

②使环境变量即时生效

[root@localhost data]# source /etc/profile

③验证是否配置成功

[root@localhost ~]# java -version

结果：

java version “1.8.0_131”
Java(TM) SE Runtime Environment (build 1.8.0_131-b11)
Java HotSpot(TM) 64-Bit Server VM (build 25.131-b11, mixed mode)

java或者javac命令也可以验证
