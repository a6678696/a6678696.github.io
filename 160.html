CentOS7安装Redis

发表于2021-06-20|更新于2021-08-04|CentOS7
|字数总计:380|阅读时长:1分钟|阅读量:

安装gcc（gcc编译c的，因为redis是c编写的，所以我们先安装下gcc）

yum install gcc-c++

wget方式下载redis压缩包，并解压，以及编译

下载

wget http://download.redis.io/releases/redis-3.2.9.tar.gz

解压

tar -zxvf redis-3.2.9.tar.gz

编译

[root@izy4gmk7hhqg1ez ~]# cd redis-3.2.9
[root@izy4gmk7hhqg1ez redis-3.2.9]# make

开始安装Redis

[root@izy4gmk7hhqg1ez redis-3.2.9]# make PREFIX=/usr/local/redis install

安装完后，需要将刚刚解压的文件夹内 redis.conf 复制到 redis 的安装目录/usr/local/redis/中

[root@izy4gmk7hhqg1ez redis-3.2.9]# cp /root/redis-3.2.9/redis.conf /usr/local/redis/

前台启动和关闭Redis

[root@izy4gmk7hhqg1ez ~]# /usr/local/redis/bin/redis-server

看到下面的界面说明已经启动成功了

按ctrl+c退出

这种启动是前端或者前台启动，假如退出 程序就终止或者退出了。

所以这种服务程序，必须后端运行；

修改配置文件redis.conf

打开配置文件（打开后按/键，并输入关键字按下回车键可快速搜索文本内容）

[root@izy4gmk7hhqg1ez ~]# vi /usr/local/redis/redis.conf

daemonize no 修改成：daemonize yes

加载配置文件运行Redis

[root@izy4gmk7hhqg1ez ~]# /usr/local/redis/bin/redis-server /usr/local/redis/redis.conf

搜索Redis服务

[root@izy4gmk7hhqg1ez ~]# ps -ef | grep -i redis

看到端口号说明已经启动成功了

停止Redis服务

[root@izy4gmk7hhqg1ez ~]# /usr/local/redis/bin/redis-cli shutdown

再次搜索Redis服务，发现端口号没了，说明Redis服务已经停止了

PS.
