Thymeleaf常用属性总结

发表于 2021-02-07 更新于 2022-04-26 分类于 前端

th:text

用于显示转义后的文本（不尊重我们的HTML标记而对其进行转义）

截取字符串

格式化时间

th:utext

用于显示未转义的文本（尊重我们的HTML标记而不对其进行转义）

th:if

条件判断，条件成立则在页面中显示，不成立则不显示

th:each

循环语句，如果要显示一个List集合的所有内容则使用循环遍历，下面代码中blogList代表要循环遍历的集合，blog则代表集合中的每个元素（或实体）

th:href

a标签的href链接、或者是HTML头文件的link引入地址

th:src

img标签的图片链接、或者是HTML头文件的script引入地址

th:title

描述了元素的额外信息 (作为工具条使用)，当页面上无法显示全部文本时，可通过title实现鼠标悬停实现查看全部文本信息

th:style

行内样式，可以结合条件表达式实现动态选择（满足条件表达式则选择第一个结果，不满足则选择第二种结果）

th:value

只需要List的部分元素

当一个集合中的元素数量超过我们需要显示的元素数量时，可使用……Stat.count和th:if（省略号处根据具体情况填写）

Java从键盘输入的三种方法

发表于 2021-02-07 更新于 2022-04-26 分类于 后端

System.in的read方法

这种方式极其简单，但是只能读入一个字符，且必须是字符类型，输出该字符的ASCII码。（如果输入字符串，则指能读入第一个字符，在下面的截图中可以看到：输入ab时输出了97，那么我们可以判断出只能读入第一个字符，因为a的ASCII码是97）

InputStreamReader和BufferedReader方法

这种方式可以读取一个字符串，但是如果需要读取int，float等类型仍需要自己转换

Scanner类

这种方式使用java5之后添加的Scanner类，Scanner类提供了读取int，float及字符串的方法，使用十分方便。同时，Scanner不仅可以读取键盘输入值，也可以读取文件内容，只需要将构造方法中的数据来源切换成该文件即可

如果先读取int，float，再读取字符串，要让指针移到下一行开头，不然读取不了字符串，反过来读取就没问题

二元运算符连接两个值

发表于 2021-01-29 更新于 2022-04-26 分类于 后端

当用一个二元运算符连接两个值时（例如a+b，a是整数，b是浮点数），先要将两个操作数转换成同一种类型，然后再进行计算

如果两个数中有一个是double类型，则另一个会转换成double类型（不满足当前条件则向下判断）

否则，如果两个数中有一个是float类型，则另一个会转换成float类型

否则，如果两个数中有一个是long类型，则另一个会转换成long类型

否则，两个数都被转换成int类型
