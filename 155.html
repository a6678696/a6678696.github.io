CentOS7安装和配置MySQL5.7

发表于2021-06-17|更新于2021-08-20|CentOS7
|字数总计:370|阅读时长:1分钟|阅读量:

安装wget

yum -y install wget

下载和安装MySQL源包

wget https://dev.mysql.com/get/mysql57-community-release-el7-11.noarch.rpm

yum -y localinstall mysql57-community-release-el7-11.noarch.rpm

在线安装MySQL

yum -y install mysql-community-server

启动MySQL服务

systemctl start mysqld

设置开机启动

[root@localhost ~]# systemctl enable mysqld
[root@localhost ~]# systemctl daemon-reload

修改root本地登录密码

mysql安装完成之后，在/var/log/mysqld.log文件中给root生成了一个临时的默认密码（下图中划红线的就是临时密码）。

[root@localhost ~]# vi /var/log/mysqld.log

进入mysql命令行，修改密码

mysql> ALTER USER ‘root‘@’localhost’ IDENTIFIED BY ‘Caofeng2012@’;

修改密码为 Caofeng2012@ (备注 mysql5.7默认密码策略要求密码必须是大小写字母数字特殊字母的组合，至少8位)

设置允许远程登录

Mysql默认不允许远程登录，我们需要设置下，并且防火墙开放3306端口；

mysql> GRANT ALL PRIVILEGES ON . TO ‘root‘@’%’ IDENTIFIED BY ‘Caofeng2012@’ WITH GRANT OPTION;

设置防火墙

[root@localhost ~]# firewall-cmd --zone=public --add-port=3306/tcp --permanent

[root@localhost ~]# firewall-cmd --reload

1配置默认编码为utf8

修改/etc/my.cnf配置文件，在[mysqld]下添加编码配置，如下所示：

[mysqld]

character_set_server=utf8

init_connect=’SET NAMES utf8’

打开my.cnf文件命令：

[root@localhost ~]# vi /etc/my.cnf

重启mysql服务

[root@localhost ~]# systemctl restart mysqld

再查看编码

mysql> show variables like ‘%character%’;
