Redis字符串

发表于2021-06-20|更新于2021-08-04|Redis
|字数总计:649|阅读时长:3分钟|阅读量:

获取指定 key 的值

get key：

127.0.0.1:6379> get name
“ledao”

设置指定 key 的值（如果key不存在就新建并复制，存在则值覆盖）

set key value

127.0.0.1:6379> set name ledao
OK
127.0.0.1:6379> get name
“ledao”
127.0.0.1:6379> set name ledao333
OK
127.0.0.1:6379> get name
“ledao333”

删除指定key

del key

127.0.0.1:6379> get num
“1”
127.0.0.1:6379> del num
(integer) 1
127.0.0.1:6379> get num
(nil)

返回 key 中字符串值的子字符（包括头包括尾，下标从0开始计算）

127.0.0.1:6379> set name ledao333
OK
127.0.0.1:6379> get name
“ledao333”
127.0.0.1:6379> getrange name 0 1
“le”

将给定 key 的值设为 value ，并返回 key 的旧值(old value)

getset key value

127.0.0.1:6379> getset name LeDao
“ledao333”
127.0.0.1:6379> get name
“LeDao”

只有在 key 不存在时设置 key 的值

setnx key value

127.0.0.1:6379> get name
“LeDao”
127.0.0.1:6379> setnx name ledao
(integer) 0
127.0.0.1:6379> get name
“LeDao”

返回 key 所储存的字符串值的长度

strlen key

127.0.0.1:6379> get name
“LeDao”
127.0.0.1:6379> strlen name
(integer) 5

同时设置一个或多个 key-value 对

mset key1 value1 key2 value2 …

127.0.0.1:6379> mset name1 ledao1 name2 ledao2
OK
127.0.0.1:6379> get name1
“ledao1”
127.0.0.1:6379> get name2
“ledao2”

将 key 中储存的数字值增一

incr key

127.0.0.1:6379> set num 0
OK
127.0.0.1:6379> get num
“0”
127.0.0.1:6379> incr num
(integer) 1
127.0.0.1:6379> get num
“1”

将 key 中储存的数字值减一

decr key

127.0.0.1:6379> get num
“2”
127.0.0.1:6379> decr num
(integer) 1
127.0.0.1:6379> get num
“1”

如果 key 已经存在并且是一个字符串， APPEND 命令将指定的 value 追加到该 key 原来值（value）的末尾，如果不存在就相当于set key value

append key value

127.0.0.1:6379> get name
“LeDao”
127.0.0.1:6379> append name 666
(integer) 8
127.0.0.1:6379> get name
“LeDao666”
127.0.0.1:6379> get name4
(nil)
127.0.0.1:6379> append name4 666
(integer) 3
127.0.0.1:6379> get name4
“666”
